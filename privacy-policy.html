Politique de confidentialité
OPNsense Mobile

Dernière mise à jour : juin 2026

OPNsense Mobile est une application non officielle permettant de gérer un firewall OPNsense depuis un appareil Android. Cette politique de confidentialité explique comment l'application traite vos données.

1. Données collectées

OPNsense Mobile ne collecte aucune donnée personnelle.

  • Pas de compte utilisateur
  • Pas d'analytics
  • Pas de tracking
  • Pas de publicité

2. Données stockées localement

Les informations suivantes sont stockées uniquement sur votre appareil, dans le stockage sécurisé Android (Android Keystore) :

  • L'URL de votre firewall OPNsense
  • Votre clé API OPNsense
  • Votre secret API OPNsense

Ces données ne quittent jamais votre appareil, sauf pour établir la connexion directe avec votre propre firewall.

3. Communications réseau

L'application communique uniquement avec le firewall OPNsense que vous avez configuré. Aucune donnée n'est envoyée à des serveurs tiers, à des services cloud ou aux développeurs de l'application.

4. Permissions Android

  • INTERNET — pour se connecter à votre firewall OPNsense

5. Sécurité

Les identifiants API sont chiffrés et stockés via le mécanisme de stockage sécurisé d'Android. La connexion avec votre firewall s'effectue en HTTPS.

6. Modifications

Cette politique peut être mise à jour occasionnellement. Toute modification sera reflétée sur cette page avec une nouvelle date de mise à jour.

7. Contact

8. Avertissement
